An end to end GenAI app powered by Groq and LangChain. Simplifies complex topics into ''Explain Like I'm 5'' summaries using Chain of Thought prompting. Features a Streamlit UI for instant, structured explanations with analogies and quizzes.
It takes complex topics and explains them in two distinct ways: **Technically** (for pros) and **Simply** (for a 5-year-old).

## üöÄ Features

* **Dual-Mode Explanations:** Generates a technical summary and a simplified analogy side-by-side.
* **Persona-Based AI:** Uses a "Kindergarten Teacher" persona (Professor Sparkle) to make learning fun.
* **Chain of Thought Reasoning:** Forces the AI to think of an analogy *before* explaining, ensuring higher quality output.
* **Blazing Fast:** Powered by **Groq's LPU** (Language Processing Unit) for near-instant inference.
* **Interactive UI:** Built with **Streamlit** for a clean, responsive user experience.
